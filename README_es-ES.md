# Material de Entrenamiento - Fundamentos de jQuery #
Este repositorio contiene material en constante evolución, que puede ser usado como material de entrenamiento en una clase de Fundamentos de jQuery, incluyendo ejercicios, soluciones y un libro web. 

Si esta buscando la ultima versión del libro, visite http://www.rebeccamurphey.com/jqfundamentals/ o descargue [el tag más reciente](http://github.com/rmurphey/jqfundamentals/downloads) y navegue a /book/release/html/. 

## Contribucion ##
Este material esta licenciado bajo la licencia Creative Commons Attribution-Share Alike 3.0 US, porque me entusiasma ver a otras personas contribuir. ¡Solicitudes de Pull son bienvenidas!

El contenido del libro esta escrito en [DocBook 5.0](http://www.docbook.org/), para lo cual es un poco difícil de escribir debido a la falta de buenos editores de XML gratuitos, pero genera todo tipo de formatos. Yo use la versión gratuita de  [XMLEditor](http://www.xmlmind.com/xmleditor/) para generar los archivos de cada capitulo. Si esta interesado en contribuir con el contenido, pero no quiere lidiar con DocBook, hágamelo saber y podemos buscar una alternativa. De la misma forma, si tiene contenido existente que puede ser bueno para incluir en el libro, hágamelo saber también. Usted será, obviamente, acreditado por cualquier contribución.

## Uso del Material ##
De acuerdo a la licencia, usted puede usar este material; si planea usarlo para dar una clase, me encantaría que me lo hiciera saber.

## Generando el HTML ##
Del directorio /book, ejecute

`./scripts/install.sh`

Despues, ejecute

`./scripts/publish.sh`

`./scripts/publish-pdf.sh`
